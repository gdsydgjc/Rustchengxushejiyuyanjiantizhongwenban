### 关于 Rust

Rust 是一种现代的系统级编程语言，由 Mozilla 研究院开发，强调速度、内存安全和并行性。它的设计哲学鼓励编写低层级代码的同时，避免常见的编程错误类型，如空指针异常、数据竞争等，通过其独特的所有权和生命周期概念来保证内存安全和并发性。
